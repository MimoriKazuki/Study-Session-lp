APPLICATION お申し込み

FAQ よくある質問

Q

プログラミング未経験でも大丈夫ですか？

+

はい、大丈夫です！生成AIを使うことで、プログラミング未経験でも簡単にゲームを作ることができます。講師が丁寧にサポートしますので、安心してご参加ください。

Q

持参するものはありますか？

+

筆記用具のみお持ちください。パソコンは会場でご用意いたします。作成したゲームはUSBメモリやクラウドで持ち帰ることができます。

Q

保護者の同伴は必要ですか？

+

小学生の参加者は、送迎をお願いいたします。教室中の同伴は必須ではありませんが、見学を希望される場合は事前にお知らせください。

Q

作ったゲームは持ち帰れますか？

+

はい、作成したゲームのデータはお持ち帰りいただけます。自宅でも継続して開発を進めることができ、自由研究の作品として活用できます。
